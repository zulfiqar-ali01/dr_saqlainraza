# Saqlain Portfolio

This project is a personal portfolio website for Dr. Saqlain Raza, showcasing his academic profile, data science services, publications, and contact information.

## Features

- **Responsive Design**: The website is designed to be fully responsive, ensuring a great user experience on both desktop and mobile devices.
- **Tailwind CSS**: Utilizes Tailwind CSS for styling, allowing for a modern and clean design.
- **TypeScript**: Built with TypeScript for better type safety and developer experience.
- **Framer Motion**: Integrates Framer Motion for smooth animations and transitions.
- **Lucide Icons**: Uses Lucide icons for a visually appealing interface.

## Installation

To get started with this project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/saqlain-portfolio.git
   ```

2. Navigate into the project directory:
   ```bash
   cd saqlain-portfolio
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to view the application.

## Usage

You can navigate through the website using the navigation bar at the top. The main sections include:

- **Academic Profile**: Information about Dr. Saqlain Raza's academic background and current position.
- **Data Science Services**: Overview of the data science services offered.
- **Publications**: A collection of publications authored by Dr. Saqlain Raza.
- **Contact**: Information on how to get in touch.
